# 🐍 Python Mini Projects

A collection of terminal-based mini projects built in Python.

---

## 📁 Projects

### 🎰 Slot Machine
A slot machine game with 5 symbols and payout multipliers.
- Bet system with balance tracking
- Symbols: 🍒 🍉 🍋 🔔 ⭐

### 🏦 Banking Program
A simple banking system.
- Deposit, withdraw, and check balance
- Input validation

### ✂️ Rock Paper Scissors
Classic rock paper scissors against the computer.

### 🔢 Number Guessing Game
Guess the randomly generated number.
- Tracks number of attempts

### 🛒 Concession Stand
A terminal-based ordering system.
- Browse items and place orders
- Calculates total bill

### ⏱️ Countdown Timer
A terminal countdown timer.
- Input custom time
- Counts down to zero

### 🧮 Calculator
Basic arithmetic calculator.

### 🌡️ Temperature Converter
Convert between Celsius, Fahrenheit, and Kelvin.

### 📝 Mad Libs
A fun word game that generates random stories.

### ❓ Quiz Game
A multiple choice quiz game.
- Tracks score

### 🛍️ Shopping Cart
A terminal shopping cart system.
- Add items and calculate total

---

## ▶️ How to Run

```bash
python filename.py
```
